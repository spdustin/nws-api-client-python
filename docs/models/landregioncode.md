# LandRegionCode

Land region code. These correspond to the six NWS regional headquarters:
* AR: Alaska Region
* CR: Central Region
* ER: Eastern Region
* PR: Pacific Region
* SR: Southern Region
* WR: Western Region



## Values

| Name  | Value |
| ----- | ----- |
| `AR`  | AR    |
| `CR`  | CR    |
| `ER`  | ER    |
| `PR`  | PR    |
| `SR`  | SR    |
| `WR`  | WR    |
