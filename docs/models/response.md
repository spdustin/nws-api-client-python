# Response

The code denoting the type of action recommended for the target audience.
This corresponds to responseType in the CAP specification.



## Values

| Name        | Value       |
| ----------- | ----------- |
| `SHELTER`   | Shelter     |
| `EVACUATE`  | Evacuate    |
| `PREPARE`   | Prepare     |
| `EXECUTE`   | Execute     |
| `AVOID`     | Avoid       |
| `MONITOR`   | Monitor     |
| `ASSESS`    | Assess      |
| `ALL_CLEAR` | AllClear    |
| `NONE`      | None        |
